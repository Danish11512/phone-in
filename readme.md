# Vite + TypeScript (minimal template)

This repository has been converted to a minimal Vite + TypeScript project without a framework. It contains simple template files for JavaScript, TypeScript, and CSS.

## Run the project

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Template files

- `src/template.ts` — TypeScript sample module with a simple DOM animation.
- `src/template.js` — JavaScript equivalent module.
- `src/style.css` — Minimal CSS styles.
